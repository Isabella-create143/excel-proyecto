MICROSOFT EXCEL



Microsoft Excel es un software de hoja de cálculo desarrollado por Microsoft, que permite organizar, analizar, calcular y visualizar datos numéricos y alfanuméricos en tablas compuestas por filas y columnas.

Funciona como una hoja de cálculo electrónica organizada en una cuadrícula de filas (números) y columnas (letras) que forman celdas, donde se introducen, ordenan y analizan datos, texto y fórmulas.

FUNCIONES EN EXCEL:

1. Función SI: Sirve para realizar comparaciones lógicas, evaluando si una condición es verdadera o falsa y devolviendo un resultado específico para cada caso.

Ejemplo para la fórmula: =SI(prueba_lógica, [valor_si_verdadero], [valor_si_falso]) .

2. Función Y: Sirve para comprobar si todas las condiciones lógicas establecidas son verdaderas. Devuelve VERDADERO si se cumplen todos los argumentos, y FALSO si al menos uno de ellos no se cumple.

Ejemplo para la fórmula: =SI(Y(valor_lógico1, [valor_lógico2], ...), [valor_si_verdadero], [valor_si_falso]) .

3. Función O: Sirve para evaluar múltiples condiciones lógicas simultáneamente y devolver VERDADERO si al menos una de ellas se cumple, o FALSO solo si todas son falsas.

Ejemplo para la fórmula: =SI(O(valor_lógico1, [valor_lógico2], ...), [valor_si_verdadero], [valor_si_falso]) .

4. EXTRAE (códigos): Sirve para obtener un número específico de caracteres del medio de una cadena de texto, empezando en la posición que tú definas.

Ejemplo para la fórmula: =EXTRAE(texto, posición_inicial, núm_de_caracteres) .

5. CONCATENAR (identificadores): Sirve para unir dos o más cadenas de texto, números o referencias de celda en una sola celda.

Ejemplo para la fórmula: =CONCATENAR(texto1, [texto2], ...) .

TABLA DINÁMICA PARA REPORTES:

Es una herramienta interactiva y potente que resume, analiza y organiza grandes volúmenes de datos automáticamente sin usar fórmulas complejas.

Ejemplo para aplicar: Insertar>Tabla dinámica; usa la "Lista de campos" para arrastrar datos a Filas/Columnas y métricas a "Valores".

GRÁFICO DINÁMICO:

Permiten visualizar, filtrar y resumir grandes volúmenes de datos de forma interactiva; se crean a partir de tablas dinámicas.

Ejemplo para aplicar: Insertar>Gráfico dinámico.

MACRO:

Son secuencias de acciones grabadas (comandos) escritas en VBA (Visual Basic para Aplicaciones) que automatizan tareas repetitivas, ahorrando tiempo en reportes, formatos o cálculos rutinarios. Se graban desde la pestaña "Programador".

Ejemplo para aplicar: Programador>Grabar Macros.
